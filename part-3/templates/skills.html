My Skills

This page demonstrates Jinja2 for loops by iterating over a Python list.

{% for skill in skills %} {{ skill }} {% endfor %}
How this works:

The skills are passed from app.py as a Python list.

skills = ['Python', 'HTML', 'CSS', 'JavaScript', 'Flask']

Skills List

    {% for skill in skills %}
  • {{ loop.index }}. {{ skill }} {% if loop.first %} (First skill) {% endif %} {% if loop.last %} (Last skill) {% endif %}
    • {% endfor %}

Jinja2 Loop Syntax

{% for skill in skills %}
    {{ skill }}
{% endfor %}

Useful Loop Variables

Variable Description
loop.index Current iteration (1-indexed)
loop.index0 Current iteration (0-indexed)
loop.first True if first iteration
loop.last True if last iteration
loop.length Total number of items
